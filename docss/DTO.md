*Базовые* поля - ID, createdAt, updatedAt, *будет дополняться* - поля, которые меняются только с серверной стороны, либо самой БД

Название | Назначение | Пример полей
*EntityBaseName*GetRequest | Входные данные от клиента | ID/*поле с конкретным значенем для поиска*
*EntityBaseName*UpdateRequest | Входные данные для обновления сущности в базе данных | Все поля?, кроме *базовых* 
*EntityBaseName*GetListRequest | Выходные данные от клиента | IDs/*поле с конкретным значением для поиска* 
*EntityBaseName*Response | Выходные данные для API | *базовые* поля + поля конкретной бизнес сущности
*EntityBaseName*UpdateResponse | Выходные данные для API | Обновленная сущность целиком
*EntityBaseName*ListResponse | Выходные данные для API | список сущностей соответствующих входным данным
